UMEWE

Whether you’re looking to meet a friend, family member, or even a date, UMEWE will find the midway point between two different locations and display a list of 10 restaurants within that radius. You can even further define your search by typing in a specific parameter such as “sushi,” and even sort by cost, distance or popularity. This project used Bing Maps, Zomato, and Geocoding APIs along with the CSS framework Bulma.

EAT-DA-BURGER

Eat-Da-Burger! is an application that lets users input the names of burgers they'd like to eat. It features a fun burger logger with MySQL, Node, Express, Handlebars and a homemade ORM. It follows the MVC design pattern. Whenever a user submits a burger's name, your app will display the burger on the left side of the page waiting to be devoured. Each burger in the waiting area also has a Devour button and a delete button. When the user clicks either, the burger will move to the right side of the page.

DAILY WORK PLANNER

This is a simple calendar application that allows a user to save events for each hour of the day. Work days can be stressful enough, so this daily work planner can help manage and organize your day. Moment.js was used to sync to real time. The greyed out portion represents past hours, purple represents the current hour, and green refers to future hours. Bootstrap CSS framework was used for this project.

JAVASCRIPT QUIZ

This is a timed code quiz with multiple-choice questions that tests your knowledge on JavaScript questions. You have 1 minute and 15 seconds to complete the quiz. If you get a wrong answer, your timer will reduce by 10 seconds. At the end you can enter your high scores and they will be saved to the local server, so anyone can compare! Bootstrap CSS framework was used for this project.

PASSWORD GENERATOR

This app creates a unique and secure password adding a better level of security. In order to produce a safe and unique password, certain criteria must be met. It has to have a length of 8-128 characters, at least one uppercase character, one lowercase character, one numeric character and one special character.
